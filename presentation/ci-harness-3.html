The evidence boundary

We can defend

  • ✓ North Mini is strong at bounded CI repair.
  • ✓ Explicit contracts sharply reduce its false-greens.
  • ✓ Maintenance is a plausible low-cost operating zone for both models.
  • ✓ Gemma 4 12B can create useful maintenance patches.
  • ✓ Hidden acceptance materially changes the merge decision.

We cannot defend yet

  • ✗ North Mini is globally better than Gemma 4 12B.
  • ✗ Product workflows are fully measured.
  • ✗ Pass@1 is a stable model characteristic.
  • ✗ Contract-visible success proves deep understanding.
  • ✗ Every shared failure is definitely a model failure.
