The hidden contract cannot be inferred from the public test alone. You can't deduce "half-up cent rounding" from seeing 0.29 × 3 = 87.

What False-Greens Actually Look Like

decimal_money_rounding

- total = 0.0
+ total = Decimal("0")
- total += float(line["unit_price"]) * int(line["quantity"])
+ total += Decimal(line["unit_price"]) * int(line["quantity"])
return int(total * 100) ← truncates, doesn't round half-up

Test: 0.29 × 3 = 87¢. Returns 87. Green. Hidden: 0.005 × 1 should be 1¢. Returns 0. Silent billing error.

Laguna produced this exact patch twice on independent attempts. Same wrong code, fresh start each time. Deterministic false-green.

audit_log_redaction

Test: redact password field. Agent replaces top-level key. Green. Hidden: nested metadata.api_key, profile.token, items[0].authorization. None redacted. Didn't recurse.

metric_semantic_mismatch

Dashboard combines cents + dollars. Agent normalizes in compute() — the raw API. Green. Hidden: compute() must return raw values. Normalization belongs in dashboard_total(). Right math. Wrong layer.
